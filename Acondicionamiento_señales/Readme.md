## Sistema de adquisición de señal con célula de carga

Desarrollo de un sistema de medida basado en una célula de carga, incluyendo el diseño del acondicionamiento de señal y la adquisición de datos para su posterior procesamiento.

La señal analógica procedente de la célula de carga es capturada mediante un dispositivo Analog Discovery 2, permitiendo la visualización y análisis de la señal en tiempo real.

El procesamiento y la interfaz de usuario han sido desarrollados en LabVIEW, donde se implementa:
- Visualización de la señal
- Filtrado y tratamiento de datos
- Conversión a magnitud física (peso)
- Interfaz gráfica para monitorización

El proyecto incluye el diseño del sistema de acondicionamiento de señal necesario para adaptar la salida de la célula de carga (baja amplitud) a niveles adecuados para adquisición.

Este proyecto destaca por:
- Diseño de electrónica analógica (acondicionamiento de señal)
- Adquisición de datos en tiempo real
- Integración hardware-software
- Desarrollo de interfaces en LabVIEW
