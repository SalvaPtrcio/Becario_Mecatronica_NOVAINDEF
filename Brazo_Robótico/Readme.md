## Brazo robótico 6 DOF con ROS2

Desarrollo completo de un brazo manipulador robótico de 6 grados de libertad como Trabajo de Fin de Grado, incluyendo diseño, integración y programación del sistema.

El robot combina dos tipos de actuadores:
- Motores Dynamixel
- Motores RDrive de Rozum Robotics

La comunicación se realiza mediante bus CAN utilizando interfaz CAN-USB, permitiendo control robusto y en tiempo real.

El sistema ha sido desarrollado íntegramente en ROS2 (C++), implementando:
- Control articular (joint space)
- Control cartesiano (end-effector space)
- Teleoperación completa mediante joystick

Este proyecto demuestra capacidades en:
- Robótica avanzada y cinemática
- Integración de hardware heterogéneo
- Sistemas distribuidos con ROS2
- Comunicaciones industriales (CAN)
