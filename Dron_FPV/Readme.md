## Dron FPV de desarrollo propio

Diseño, ensamblaje y configuración completa de un dron FPV construido desde cero, incluyendo soldadura de todos los componentes electrónicos e integración del sistema de vuelo.

El sistema está basado en controladora configurada mediante Betaflight, con transmisión de vídeo digital utilizando DJI Air Unit O4 Pro y gafas FPV, permitiendo operación en tiempo real con baja latencia.

El dron utiliza baterías LiPo 6S y cuenta con electrónica de potencia optimizada para alto rendimiento. La comunicación se realiza mediante radiofrecuencia, alcanzando distancias de operación de hasta 5 km en condiciones óptimas.

Este proyecto destaca por la integración completa de:
- Electrónica de potencia (LiPo, ESCs)
- Comunicaciones RF (control y vídeo)
- Montaje y soldadura de hardware
- Configuración de sistemas de vuelo reales (PID)
