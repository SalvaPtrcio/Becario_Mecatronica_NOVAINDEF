## Robot móvil diferencial autónomo

Diseño y construcción completa de un robot móvil con tracción diferencial, desarrollado desde cero tanto a nivel hardware como software.

El sistema está basado en Arduino con programación en C++ , MicroROS y gestión de tareas mediante FreeRTOS, permitiendo ejecución concurrente y control en tiempo real.

El robot integra:
- Sensores ultrasónicos (sonar) para detección de obstáculos
- Motores DC para locomoción diferencial
- Modos de operación tanto manual (teleoperación) como autónomo

Se han implementado funcionalidades de:
- Navegación básica autónoma con esquiva de obstáculos
- Control remoto en tiempo real
- Arquitectura modular para control y sensado

Este proyecto destaca por:
- Integración completa de electrónica, control y mecánica
- Programación embebida en tiempo real
- Desarrollo de algoritmos de navegación reactiva
